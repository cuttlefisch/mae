#+title: Scheme: *buffer-list*
#+filetags: :scheme:api:variable:

*Type:* list of (index name kind modified?)

Information about all open buffers.

This is a read-only variable injected from editor state before each Scheme evaluation. Access it directly by name in your Scheme code.

** Example
#+begin_src scheme
(message (string-append "Buffer: " *buffer-list*))
#+end_src
