#+title: Scheme: set-local-option!
#+filetags: :scheme:api:options:

** Signature
#+begin_src scheme
(set-local-option! KEY VALUE)
#+end_src

Set a buffer-local option on the active buffer.

** Example
#+begin_src scheme
(set-local-option! "word_wrap" "true")
#+end_src
