#+title: Scheme: cursor-goto
#+filetags: :scheme:api:navigation:

** Signature
#+begin_src scheme
(cursor-goto ROW COL)
#+end_src

Move the cursor to absolute position (0-indexed).

** Example
#+begin_src scheme
(cursor-goto 0 0) ; go to top-left
#+end_src
