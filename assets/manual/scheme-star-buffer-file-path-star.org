#+title: Scheme: *buffer-file-path*
#+filetags: :scheme:api:variable:

*Type:* string

File path of the active buffer, or empty if unsaved.

This is a read-only variable injected from editor state before each Scheme evaluation. Access it directly by name in your Scheme code.

** Example
#+begin_src scheme
(message (string-append "Buffer: " *buffer-file-path*))
#+end_src
