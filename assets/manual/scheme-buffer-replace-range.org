#+title: Scheme: buffer-replace-range
#+filetags: :scheme:api:buffer-editing:

** Signature
#+begin_src scheme
(buffer-replace-range START END TEXT)
#+end_src

Replace characters from START to END with TEXT.

** Example
#+begin_src scheme
(buffer-replace-range 0 5 "new")
#+end_src
