#+title: Scheme: buffer-delete-range
#+filetags: :scheme:api:buffer-editing:

** Signature
#+begin_src scheme
(buffer-delete-range START END)
#+end_src

Delete characters from byte offset START to END.

** Example
#+begin_src scheme
(buffer-delete-range 0 10)
#+end_src
