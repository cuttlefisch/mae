#+title: Scheme: define-key
#+filetags: :scheme:api:keymaps:

** Signature
#+begin_src scheme
(define-key MAP KEY COMMAND)
#+end_src

Bind KEY in keymap MAP to COMMAND.

** Example
#+begin_src scheme
(define-key "normal" "g g" "goto-first-line")
#+end_src
