#+title: Scheme: list-directory
#+filetags: :scheme:api:file-io:

** Signature
#+begin_src scheme
(list-directory PATH)
#+end_src

Return a list of (name is-dir?) pairs for entries in PATH.

** Example
#+begin_src scheme
(list-directory "/tmp")
#+end_src
