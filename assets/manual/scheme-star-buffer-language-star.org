#+title: Scheme: *buffer-language*
#+filetags: :scheme:api:variable:

*Type:* string

Detected language of the active buffer (e.g. "rust", "text").

This is a read-only variable injected from editor state before each Scheme evaluation. Access it directly by name in your Scheme code.

** Example
#+begin_src scheme
(message (string-append "Buffer: " *buffer-language*))
#+end_src
