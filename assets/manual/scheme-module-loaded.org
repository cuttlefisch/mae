#+title: Scheme: module-loaded?
#+filetags: :scheme:api:modules:

** Signature
#+begin_src scheme
(module-loaded? NAME)
#+end_src

Return #t if the named module is currently loaded.

** Example
#+begin_src scheme
(module-loaded? "org") ; => #t
#+end_src
