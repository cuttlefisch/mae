#+title: Git Status (Magit-lite)
#+filetags: :git:workflow:

The **Git Status** buffer (`*git-status*`) is a high-fidelity "porcelain" UI inspired by Emacs Magit. It allows you to manage your repository state without leaving the editor.

## Multi-Level Fold
Press `TAB` on a section header, file entry, or hunk header to fold/unfold that level independently. Collapse indicators (`▸`/`▾`) show fold state.

## Context-Aware Dispatch
`s`/`u`/`x` operate based on cursor position:
- **On a diff hunk/line**: stage/unstage/discard that hunk.
- **On a file entry**: stage/unstage/discard the whole file.
- **On a section header**: stage/unstage all files in that section.

## Inline Diff
Press `TAB` on a file entry to expand/collapse its inline diff. Each hunk can be further folded independently.

## Workflow
1. Open status via `SPC g s`.
2. Navigate with `j`/`k`, jump hunks with `n`/`p`.
3. Stage files/hunks with `s`.
4. Commit with `c c` (opens a commit message buffer).
