#+title: Scheme: file-exists?
#+filetags: :scheme:api:file-io:

** Signature
#+begin_src scheme
(file-exists? PATH)
#+end_src

Return #t if PATH exists on disk.

** Example
#+begin_src scheme
(file-exists? "/tmp/test.txt")
#+end_src
