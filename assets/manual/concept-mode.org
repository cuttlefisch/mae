#+title: Mode
#+filetags: :data-model:modal-editing:

MAE is **modal** like Vim. The current [[references:concept:mode][Mode]] determines which keymap is active.

** Modes
- **Normal** -- movement and commands (default).
- **Insert** -- literal text entry.
- **Visual(Char|Line)** -- selection.
- **Command** -- `:command` line.
- **Search** -- `/` incremental search.
- **ConversationInput** -- typing into the AI prompt.
- **FilePicker** -- fuzzy file open overlay.
- **ShellInsert** -- raw keyboard passthrough to [[references:concept:terminal][embedded terminal]].

Mode transitions are commands -- see [[cmd:enter-normal-mode]], [[cmd:enter-insert-mode]], [[cmd:enter-command-mode]]. The AI agent can trigger them too (that's the point of [[references:concept:ai-as-peer][The AI as Peer Actor]]).
