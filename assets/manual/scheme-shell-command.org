#+title: Scheme: shell-command
#+filetags: :scheme:api:shell:

** Signature
#+begin_src scheme
(shell-command CMD)
#+end_src

Execute CMD in a subprocess and return its stdout as a string.

** Example
#+begin_src scheme
(shell-command "date") ; => "Mon May 12 ..."
#+end_src
