#+title: Scheme: string-downcase
#+filetags: :scheme:api:string-utils:

** Signature
#+begin_src scheme
(string-downcase STR)
#+end_src

Return STR converted to lowercase.

** Example
#+begin_src scheme
(string-downcase "HELLO") ; => "hello"
#+end_src
