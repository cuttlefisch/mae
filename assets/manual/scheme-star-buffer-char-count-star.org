#+title: Scheme: *buffer-char-count*
#+filetags: :scheme:api:variable:

*Type:* integer

Total characters in the active buffer.

This is a read-only variable injected from editor state before each Scheme evaluation. Access it directly by name in your Scheme code.

** Example
#+begin_src scheme
(message (string-append "Buffer: " *buffer-char-count*))
#+end_src
