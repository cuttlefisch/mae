#+title: Scheme: check-deprecated
#+filetags: :scheme:api:modules:

** Signature
#+begin_src scheme
(check-deprecated NAME)
#+end_src

Check if NAME is deprecated. Returns the replacement name or #f.

** Example
#+begin_src scheme
(check-deprecated "old-fn") ; => "new-fn"
#+end_src
