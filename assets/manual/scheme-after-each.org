#+title: Scheme: after-each
#+filetags: :scheme:api:testing:

** Signature
#+begin_src scheme
(after-each HOOK-FN)
#+end_src

Register a teardown function for the current describe scope. Called after each it-test.

** Example
#+begin_src scheme
(after-each (lambda () (kill-buffer-by-name "*test*")))
#+end_src

*Category:* testing
