** Lesson 8: Terminal

Prerequisites: [[requires:lesson:lsp][Lesson 7: LSP]]

MAE embeds a full [[teaches:concept:terminal][terminal emulator]].

*** Commands
  ~SPC o t~ — open terminal
  ~Ctrl-\ Ctrl-n~ — exit terminal to Normal mode
  ~SPC e s~ — send current line to terminal
  ~SPC e S~ — send selection to terminal

*** Features
- Full VT100 support (vim, less, top, fzf all work)
- MCP bridge: AI agents in the terminal can call back into MAE
- Shell CWD tracking via ~/proc~
