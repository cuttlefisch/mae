#+title: Scheme: keymap-bindings

** Signature
#+begin_src scheme
(keymap-bindings MAP-NAME)
#+end_src

Return a list of (key command) pairs for a keymap.

** Example
#+begin_src scheme
(keymap-bindings "normal")
#+end_src
