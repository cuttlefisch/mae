** Lesson 4: Files & Buffers

Prerequisites: [[lesson:editing?rel=requires][Lesson 3: Editing]]

A [[concept:buffer?rel=teaches][buffer]] is the unit of editable content in MAE.

*** File commands
  ~:w~ — [[cmd:save][save]] file
  ~:e <file>~ — open file
  ~:q~ — quit (fails if unsaved)
  ~:wq~ or ~:x~ — save and quit

*** Leader shortcuts
  ~SPC f f~ — find file (fuzzy picker)
  ~SPC f d~ — file browser
  ~SPC f t~ — file tree sidebar
  ~SPC b b~ — switch buffer (palette)
  ~SPC b d~ — close buffer
