#+title: Leader Key
#+filetags: :terminology:

The key that opens MAE's which-key command menu (the **leader keypad**). In the
`doom` flavor it's ~SPC~ from Normal mode; in the `nonmodal` flavor it's ~C-;~
from Insert. Inspired by Doom Emacs / Meow's keypad. The menu itself is the same
shared tree in either flavor: ~f f~ = find file, ~b s~ = save buffer, ~h~ = help,
~a~ = AI agent, etc. The bindings live in the shared ~leader~ keymap (not under an
~SPC~ prefix); the leader key just enters a transient keypad layer that resolves
one command and returns you to editing.

See also: [[concept:keymap-flavors]].
