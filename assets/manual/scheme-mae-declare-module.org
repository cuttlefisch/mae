#+title: Scheme: mae-declare-module!
#+filetags: :scheme:api:modules:

** Signature
#+begin_src scheme
(mae-declare-module! NAME . FLAGS)
#+end_src

Declare a module to be loaded by the module system. FLAGS are optional feature flags.

** Example
#+begin_src scheme
(mae-declare-module! "org" '((+journal . #t)))
#+end_src
