#+begin_src scheme
(ai-tool-param! TOOL-NAME PARAM-NAME TYPE DESCRIPTION REQUIRED)
#+end_src

Add a parameter definition to a registered AI tool.

** Example
#+begin_src scheme
(ai-tool-param! "my-tool" "query" "string" "Search query" #t)
#+end_src
