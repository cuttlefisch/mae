#+title: Scheme: string-contains?
#+filetags: :scheme:api:string-utils:

** Signature
#+begin_src scheme
(string-contains? HAYSTACK NEEDLE)
#+end_src

Return #t if HAYSTACK contains NEEDLE.

** Example
#+begin_src scheme
(string-contains? "hello world" "world") ; => #t
#+end_src
