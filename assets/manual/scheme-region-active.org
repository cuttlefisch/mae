#+title: Scheme: region-active?
#+filetags: :scheme:api:navigation:

** Signature
#+begin_src scheme
(region-active?)
#+end_src

Return #t if a visual selection (region) is currently active.

** Example
#+begin_src scheme
(when (region-active?) (message "Selection active"))
#+end_src
