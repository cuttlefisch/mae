#+title: Scheme: module-version
#+filetags: :scheme:api:modules:

** Signature
#+begin_src scheme
(module-version NAME)
#+end_src

Return the version string of a loaded module, or #f.

** Example
#+begin_src scheme
(module-version "org") ; => "0.1.0"
#+end_src
