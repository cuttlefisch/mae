#+title: Scheme: string-join
#+filetags: :scheme:api:string-utils:

** Signature
#+begin_src scheme
(string-join LIST SEPARATOR)
#+end_src

Join a list of strings with SEPARATOR.

** Example
#+begin_src scheme
(string-join '("a" "b" "c") ",") ; => "a,b,c"
#+end_src
