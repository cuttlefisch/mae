#+title: BufferView Enum
#+filetags: :data-model:core:

The **BufferView** enum (`buffer_view.rs`) stores mode-specific state on `Buffer`. Variants: `Conversation`, `Help`, `Debug`, `GitStatus`, `Visual`, `FileTree`, `None`.

Accessor methods: `buf.conversation()`, `buf.kb_view()`, `buf.git_status_view()`, etc. Each returns `Option<&T>` (or `Option<&mut T>` for the `_mut` variant).

This replaced 6 `Option<T>` fields that were always mutually exclusive.
