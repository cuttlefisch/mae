# Module: registers

Named registers, yank ring, and clipboard integration.

## Info

| Field | Value |
|-------|-------|
| Category | editor |
| Version | 0.1.0 |
| Dependencies | None |

## Keybindings

| Key | Command | Description |
|-----|---------|-------------|
| `"` (normal) | `prompt-register` | Select register for next yank/delete/paste |
| `"` (visual) | `prompt-register` | Select register for next yank/delete/paste |
| `SPC r r` | `show-registers` | Show register contents |
| `SPC r y` | `paste-from-yank` | Paste from yank ring |

## Configuration

No module-specific options.
