# Module: lookup

Definition lookup — dumb-jump fallback, online docs, man pages.

## Info

| Field | Value |
|-------|-------|
| Category | tools |
| Version | 0.1.0 |
| Dependencies | None |

## Keybindings

| Key | Command | Description |
|-----|---------|-------------|
| `SPC s o` | `lookup-online` | Search online documentation for word at point |
| `SPC s m` | `lookup-man` | Open man page for word at point |
