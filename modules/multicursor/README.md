# Module: multicursor

Multiple cursors — add above/below, word match, align.

## Info

| Field | Value |
|-------|-------|
| Category | editor |
| Version | 0.1.0 |
| Dependencies | None |

## Keybindings

| Key | Command | Description |
|-----|---------|-------------|
| `SPC m j` | `mc-add-cursor-below` | Add cursor on line below |
| `SPC m k` | `mc-add-cursor-above` | Add cursor on line above |
| `SPC m d` | `mc-add-at-next-word` | Add cursor at next occurrence of word |
| `SPC m a` | `mc-add-all-word` | Add cursors at all occurrences of word |
| `SPC m s` | `mc-skip-next` | Skip next occurrence |
| `SPC m c` | `mc-clear` | Clear all extra cursors |
| `SPC m l` | `mc-align` | Align cursors to column |

## Configuration

No module-specific options.
