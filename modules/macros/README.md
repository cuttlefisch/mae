# Module: macros

Keyboard macro recording and replay — q, @.

## Info

| Field | Value |
|-------|-------|
| Category | editor |
| Version | 0.1.0 |
| Dependencies | None |

## Keybindings

| Key | Command | Description |
|-----|---------|-------------|
| `q` | `start-recording-await` | Start recording macro into register (prompts for letter) |
| `@` | `replay-macro-await` | Replay macro from register (prompts for letter; `@@` repeats last) |

## Configuration

No module-specific options.
